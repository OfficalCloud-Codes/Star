# Installing
Installing Star* is quite easy.

Currently Star* is not its own yet, like Lua. To use Star* you must have at least Lua 5.5.0. You can download Lua from [here](https://www.lua.org/download.html)

After that, all you need to do is download the release from the [GitHub Repo](https://github.com/OfficalCloud-Codes/Star), Place it in the same folder and to start running `.star` files, run this:

`lua star.lua test.star`

`lua star.lua` is required. After that you may type the name of the file you would like to run.
